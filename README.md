# Parcel sample projects

Parcel build examples for...

- VanillaJS
- jQuery
- React
- preact
- Vue.js

## Installation

```bash
$ cd parcel-jquery # on jQuery
$ cd parcel-preact # on preact
$ cd parcel-react # on React
$ cd parcel-vanilla # on Vanilla
$ cd parcel-vue # on Vue

$ yarn
$ yarn start # Start server
```
